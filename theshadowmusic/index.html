Frequently Asked Questions

1. What Is The Default Prefix For the Bot?

The default prefix for the bot is: /

2. Why i can't add TheShadowMusic to my server?

TheShadowMusic is a private bot made for partner servers

3. What is The purpose of the Dashboard?

The Dashboard is a tool for admins to manage the songs that are being played in the server, users can use it to see statistics about the bot

4. Why the statistics on the Dashboard don't seem right?

The statisics shown in the Dashboard are resetted after a bot restart, these restarts can happen at any time

5. Why the bot is offline?

The bot can be offline for a different reasons; from an accident to a maintance, you can track the bot status on the official SGT Bots Statuspage

6. Why when i access the Dashboard i get an Internal Server Error?

Most of the times this means that you are currently logged in an invalid session, to fix this problem just logout by clicking here if this dosen't fix your problem please contact us

7. Why are you transitioning to Slash Commands?

There are various reasons, but the most important one is to be compliant with Discord's new design for bots: learn more here, and to make the bot as accessible as possible

8. When i will not be able to use prefix commands?

You are not going to be able to use prefix commands from the 2.0.0 update (19/09/22)

9. Will i be able to use the aliases that i am used to?

For tecnical limitations and due to the nature of Slash commands aliases are not going to be available, but you can use Discord's auto completion for commands
