TheShadowMusic Versione 2.3.0

Feature Update

Utilità per admin avanzate

Ora gli admin dei server possono gestire svariati parametri di TheShadowMusic da un singolo file di configurazione, inoltre è stato aggiunto il supporto per SGT-AdminTools

Aggiunto il comando /guildleave

Questo comando è inteso per la rimozione forzata del bot in caso di mancati permessi, questo comando è esclusivo al utilizzo dagli admin di SGT Bots

Più coerenza

Ora quando il bot indica un timestamp il formato è standard tra tutte le interazioni

Risolti Bug

Sono stati risoli svariati problemi

Aggiornato il codebase

È stato aggiornato il codebase di TheShadowMusic ad 9ac82e1

Questo sarà l'ultimo feature update del anno, Buone Feste dal team di SGT Bots
